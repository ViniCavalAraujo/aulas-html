Diferença entre id e class

Neste subtítulo estou usando id

O id é um identificador no HTML, sua representação em estilo é o "#" famoso jogo da velha ou hashtag.
Não podemos utilizar um mesmo id em mais de um elemento no mesmo documento HTML.


Já neste, estou usando class

Neste parágrafo, estou utilizando a class "principal".
Já com class, podemos utilizar a mesma class em mais de um elemento no mesmo documento HTML.
